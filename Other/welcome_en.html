Setup

This section covers the basic setup of a connection in dreaMote.

To make configuring dreaMote easier, it is able to do a semi-automatic configuration by itself. If dreaMote is not yet configured, a dialog as seen in the screenshot above appears.
If only one receiver is found on your network, the data is automatically transferred into the configuration dialog afterwards.
If more were found, you are presented with a dialog to choose which one to add.

On Enigma2-based receivers, you might need to install "enigma2-plugin-extensions-bonjour" to be able to find it properly.
Please refer to the manual of your receiver on how to do so.

In case that automatic configuration does not work, or you want to change any settings, you are free to do so using the regular configuration dialog as explained below.
  • Section Remote Host:
    • Name: an arbitrary description
    • Address: IP-Address or Hostname of receiver
    • Port: If you changed the port your receiver listens on you have to enter it here
    • Use SSL: If you use an encrypted connection you should set this to ON
  • Section Credential:
    • Username: If authentication is required, enter the username here
    • Password: If authentication is required, enter the password here
  • Section Remote Box Type:
    • Connector Type: Select this row to change the connection type (Enigma, Enigma2, Neutrino, SVDRP)
    • Now/Next in Servicelist: Show current and next event, disabled by default for performance reasons
    • Single Bouquet: If you are only using the "Favourites"-Bouquet you can set this to ON to skip Bouquet selection
    • Advanced Remote: Show additional buttons in the full remote control emulator
Each option also contains an example (and often default) value that you can use if in doubt.

Simple Remote Control

This section explains the "Simple Remote" of dreaMote.

The simple remote is an easier to use alternative to the "Full" remote control emulator which displays all available buttons. Swipe the cross to emulate an arrow key press and double tap it to emulate the OK button.
You can change to the "Full" remote control emulator at all times by going to the settings.

Reloading Data

In this section we explain how to refresh data in dreaMote.

Most views will automatically reload data as soon as they are reshown. Nevertheless there is a way to force re-requesting data from the connected receiver using the so-called "Pull to Refresh" functionality.
If you pull a list above its first entry you will start seeing something similar to the picture above. Once you pulled far enough the arrow will rotate and the text will change to "Release to refresh...".
As soon as you let go the data will be reloaded.

Screenshots

To save a screenshot taken with the application–to do so just open the emulated remote control and select the "Screenshot"-Button–you have have iOS 3.2 or newer installed on your device.
Holding your finger for one second on the picture should bring up a dialog asking you if you want to save the picture.
You find saved screenshots in the Camera Roll.
