Cette section couvre la configuration basique d'une connexion dans dreaMote.
  • Section Information serveur:
    • Nom: Une description de votre récepteur
    • Addresse: Adresse IP ou adresse DNS
    • Port: Si vous avez changé le port votre récepteur vous doivez l'entrer ici
    • Utiliser SSL: Si vous utilisez une connexion sécurisée vous devez mettre cela sur ON
  • Section Authentificatien:
    • Nom d'utilisateur : Si l'authentification est exigée, entrer le nom d'utilisateur ici
    • Mot de passe: Si l'authentification est exigée, entrer le mot de passe ici
  • Section Type de récepteur:
    • Type de connexion : Choisissez cette rangée pour changer le type de connexion (Enigma, Enigma2, Neutrino, SVDRP)
    • EPG dans bouquets : Montrez l'EPG actuel et suivant, mis sur OFF par défaut pour des raisons de performance
    • Bouquet simple : Si vous utilisez seulement le bouquet "Favourites" vous pouvez mettre sur ON pour passer la sélection des bouquets
    • Télécommande avancée : Voir tous les boutons de la télécommande
Chaque option contient aussi une valeur d'exemple (et manquer souvent à ses engagements) que vous pouvez employer dans le doute.

Cette section explique la "Télécommande Simple" de dreaMote.

La télécommande simple est une alternative plus facile à utiliser que la télécommande avancée qui montre tous les boutons disponibles. Frappez à la volée la croix pour imiter l'appui d'une touche de votre télécommande et frappez deux fois pour simuler le bouton OK.
Vous pouvez changer pour mettre la télécommande avancée à tout moment en allant dans la configuration.

Dans cette section nous expliquons comment rafraîchir les données dans dreaMote.

La plupart des vues rechargeront automatiquement les données aussitôt que l'on les ouvrent. Néanmoins il y a une façon de forcer le rechargement de données du récepteur connecté en employant la fonctionnalité appelée "Descendez pour mettre à jour".
Si vous tirez une liste, au-dessus de sa première entrée vous commencerez à voir quelque chose semblable à l'image ci-dessus. Une fois que vous avez tiré loin assez la flèche tournera et le texte changera "Relâchez pour mettre à jour".
Aussitôt que vous lâchez les données seront rechargées.
