Sie starten diese Version von dreaMote zum ersten Mal.

Für den Fall, dass Ihnen die Änderungen dieser Version nicht bekannt sind, möchten wir hier auf ein paar wichtige aufmerksam machen:
  • Verschiedene Designs (vier Varianten für den Einstieg)
  • Mehrfachauswahl für Timer- und Filmliste
  • Bestätigung in Erfolgsfällen hinzugefügt
  • Halten einer Sendung im EPG erlaubt es einen Timer hinzuzufügen
  • Mehrere Festplatten können in "Über den Receiver" angezeigt werden
  • Enigma2:
    • Support für das WebIinterface 1.7.0
    • Neue AutoTimer-Versionen werden besser unterstützt
    • Tags von Timern können bearbeitet werden
    • Einstellungen des AutoTimer könenn mit aktuellem Plugin bearbeitet werden
    • Filme können nach Tags gefiltert werden
    • Filme können verschoben werden (erfordert WebInterface 1.7.0)
    • Verbesserte Performance vieler Tabellen
